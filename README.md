# Product Manager

## Getting Started

### Prerequisites

Make sure :-
* NodeJs is installed.
* Internet Connection

### Steps 

```
git clone https://github.com/sumya01/Product_Manager.git
```
#### Start Backend Service
```
cd Product_Manager/api/
```
```
npm install && npm start
```
#### Start Frontend Service

Open New Terminal

```
cd Product_Manager/client/
```
```
npm install && npm start
```
